# Wedding Event Manager

## Overview
The Wedding Event Manager is a web application designed to help users manage wedding events efficiently. It provides features for guest registration, food and drink preferences, budget tracking, and event timelines, ensuring a smooth planning experience.

## Features
- **Guest Registration**: Easily add and manage guest details, including their food and drink preferences.
- **Food and Drink Preferences**: Customize menu options for guests based on their dietary needs.
- **Budget Tracking**: Keep track of expenses and manage the overall wedding budget effectively.
- **Event Timelines**: Organize and display important event timelines to ensure everything runs on schedule.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Modify the `styles.css` file to customize the appearance of the website.
4. Update the `script.js` file to add or modify functionality as needed.
